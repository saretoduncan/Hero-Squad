# Hero Squad

This an app that allows you to recruit a well-balanced team of superheroes (or any other character-type of your choosing)

## Built With

- JAVA 
- J-UNIT5
- GRADLE
- JAVA SPARK



## Getting Started

To get a local copy, follow these simple example steps.

### Prerequisites

A computer with a working and up to date web browser.

### Setup

- Clone the repository to your folder of choice using Git (or just download all the files)
- Install JVM 
- Install Java IDE of your choice

```
$ git clone https://github.com/saretoduncan/Hero-Squad.git

```
